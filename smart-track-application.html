SmartTrack Application

SmartTrack is a real-time school transport tracking system designed to ensure safe and transparent travel for students. I developed the mobile applications using Flutter, creating dedicated interfaces for both drivers and parents, while the admin dashboard was built with Vue.js to manage and monitor transport activities efficiently.

The backend was developed in Node.js, featuring robust APIs and real-time event handling. Authentication was implemented using Firebase OTP via RCS and SMS fallback, ensuring secure and reliable access for all user types.

SmartTrack also includes a smart notification system — when a driver starts the school route, parents receive instant alerts, keeping them informed in real-time. Data such as trip start time, student pickups, and route progress are synced continuously across devices for complete visibility and peace of mind.

The system was designed with scalability, responsiveness, and user experience in mind, providing a reliable solution for school administrators and parents alike.
